#  Temperature Converter

A simple and responsive **Temperature Converter** web application built using **HTML, CSS, and JavaScript**.  
It allows users to convert temperatures between **Celsius and Fahrenheit** with an easy page switch.

---

##  Features

-  Convert **Celsius → Fahrenheit**
-  Convert **Fahrenheit → Celsius**
-  Switch button to toggle between conversions
-  Accurate results (rounded to 2 decimal places)
-  Reset button to clear input and result
-  Works perfectly on **GitHub Pages**

---

##  Technologies Used

- **HTML** – structure of the pages  
- **CSS** – styling and layout  
- **JavaScript** – logic and interactivity  

---

├── style.css # Styling
└── README.md


---

##  How It Works

1. The project starts with `index.html` (Celsius → Fahrenheit).
2. User enters a temperature and clicks **Convert**.
3. Result is displayed instantly.
4. Clicking **Switch** navigates to the other conversion page.
5. Navigation is handled using relative paths (`./index.html`, `./ftoc.html`) for GitHub Pages compatibility.
